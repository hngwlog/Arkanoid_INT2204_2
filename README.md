# Arkanoid

A polished Arkanoid/Breakout-style game written in Java + JavaFX for an Object-Oriented Programming course project. It features single-player, local two‑player mode, animated power‑ups, persistent high scores, and a clean screen‑manager architecture.

## Features
### Core Gameplay
- Classic Arkanoid mechanics: paddle, ball, brick destruction.
- Smooth frame-based game loop implemented using AnimationTimer.
- Dynamic collision detection between balls, bricks, bosses, and paddles.
- Multiple levels with distinct map layouts loaded via MapLoader.

### Power-Ups
- AddBallPowerUp: spawns an extra ball.
- ExtendPaddlePowerUp: increases paddle size for a limited duration.
- ImmortalPowerUp: makes all balls unbreakable for a short time.

### Game Modes
- Single Player Mode: progress through multiple levels.
- Multiplayer Mode: two players compete side-by-side.
- High Score System with JSON persistence.
- Pause / Resume System and Configurable Settings.

### Multimedia
- Integrated AudioManager for background music and sound effects.
- Custom Cyberpunk pixel font applied globally.
- Animated sprite sheets for power-ups, bosses, etc.
